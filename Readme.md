# Meme-hub

A modern platform for creating memes together with your friends.

## 🚀 Features

- Upload Image and create memes
- Responsive design for both desktop and mobile


## 🛠️ Technologies

- Frontend: React.js
- Backend: Node.js, Express
- Styling: CSS
- socket.io for real-time updates
